# Hey
Project created with [Meteor Boilerplate](https://github.com/2Associes/meteor-boilerplate)

## Getting started

### Installation

First run:

	meteor npm install

Then you can run the app with:

	meteor npm run dev

### Tests

Run tests with:


	meteor npm run test
	meteor npm run test-watch


### Usefull

To reset project use

	meteor reset
